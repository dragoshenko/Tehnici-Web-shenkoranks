# shenkoRanks

![shenkoRanks Logo](shenkoRanksLogo.png)

## Description

shenkoRanks is a web application that provides rankings for shenko's (me) top artists and albums of all time. It offers insights into artists and albums from both the United States and Romania. The application also features a working media player for users to enjoy curated playlists.

## Features

- **Top Artists All-Time**: Explore rankings of top artists from the United States and Romania.
- **Top Albums All-Time**: Discover rankings of top albums from the United States and Romania.
- **Contact**: Connect with the developer via Instagram for inquiries or suggestions.
- **Media Player**: Enjoy curated playlists directly within the application.

## Usage

Simply navigate through the menu to access different sections of the application:
- Click on "Home" to return to the homepage.
- Explore "Top Artists All-Time" and "Top Albums All-Time" to view rankings for different regions.
- Click on "Contact" to connect with the developer via Instagram.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Spotify Embed API
